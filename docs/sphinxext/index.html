Sphinx extensions¶

Package Contents¶

The default_profile.sphinxext package.

Imports the modules found in the current directory and utilizes pkgutil and pkgutil.extend_path() to extend the packages __path__ parameter.

It only imports the modules below if this repository has been installed.

If this weren’t true, then starting IPython without this package installed would emit ImportError on startup, which would be frustrating for users.
