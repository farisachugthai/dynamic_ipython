04_easy_import¶

This module simply populates the IPython namespace with frequently used modules.

A check is run to see if the user is running a new enough version of Python to utilize the python package pynvim instead of neovim.

easy_import(mod)[source][source]¶

Import a module.

Utilizes importlib.import_module() from importlib so that any valid alphanumeric string can be passed without crashing the interpreter.

Caution

Raising ImportError not ModuleNotFoundError may render the try/catch useless on python3.7>=.

Parameters

mod (str) – A module to import.

Returns

ret_mod – Imported module. Specifically used for neovim in this instance but can be interactively used for any module the user needs to import.

Return type

mod

Raises

ImportError – If the module isn’t imported.
