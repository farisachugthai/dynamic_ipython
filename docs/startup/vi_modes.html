32_vi_modes module¶

Dynamically add keybindings to IPython.

Keybindings and Toggling Insert Mode¶

URL

https://ipython.readthedocs.io/en/stable/config/details.html#keyboard-shortcuts

Effectively adds j k as a way to switch from insert mode to normal mode, or as prompt_toolkit calls it, “navigation mode”.

Also displays how to integrate prompt_toolkit and IPython together well.

Ultimately this module hopes to implementing Tim Pope’s vim plugin rsi in IPython.

Specifically it intends on adding the standard readline bindings to Vim’s insert mode.

See also

prompt_toolkit.key_binding.defaults

str (path) Has all the named commands implemented here and possibly more.
