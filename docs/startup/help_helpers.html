06_Help_Helpers¶

Creates 2 functions in userspace to save the output of help(python_object) to a file or output to sys.stdout.

06_help_helpers module¶

Export functions to redirect help() output.

This module utilizes the examples given in the official documentation for contextlib.

In addition to utilizing contextlib, we create a function that allows that allows searching through the members of an object.

This is intended to be exposed to the user as a quick, interactive tool with an easier “grep-like” interface for objects that are too large to be quickly and easily understood based on the output of dir().

print_help(arg=None)[source][source]¶

Redirect help() to sys.stderr.

save_help(arg=None, output_file=<colorama.ansitowin32.StreamWrapper object at 0x00000230861ECDA0>)[source][source]¶

Write help() to a file. Defaults to sys.stdout.

page_help(arg=None)[source][source]¶

Using IPython’s pinfo magic to page information to the console.

Also noting it’s the only function in this module that actually needs the IPython instance so the imports were moved here.
