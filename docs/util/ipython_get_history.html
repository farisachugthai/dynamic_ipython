ipython_get_history¶

Extract a session from the IPython input history.

Usage¶

If ‘outputfile’ is not given, the relevant history is written to sys.stdout. If ‘outputfile’ has a .py extension, the translated history (without IPython’s special syntax) will be extracted.

Examples¶

python3 ipython-get-history.py 57 record.ipy

This script is a simple demonstration of IPython.core.history.HistoryAccessor. It should be possible to build much more flexible and powerful tools to browse and pull from the history database.

So this is a rewrite of that history accessor since IPython’s handling of history sessions out of the box is oddly limited.

Let’s ignore the direct calls to sys.argv and combine argparse and the magic_argparse functions to make something more durable and useful.
