Module contents¶

They’re still all useful in their current state and when the user is running IPython interactively, but none of the scripts havet been fleshed out to the extent that they could be easily changed to IPython extensions.

Goals¶

Currently the module aims to:

  1. Create consistent logging.Logger objects

  2. Make a better pager on Windows

  3. Create a collection of classes that can more easily remove platform-specific issues that continue to arise.

In addition, configure a module-wide logger by equating logging.BASIC_FORMAT with a pre-determined template string like so:

>>> logging.BASIC_FORMAT = '%(asctime)s : %(levelname)s : %(message)s'
>>> UTIL_LOGGER = logging.getLogger('default_profile.util')
>>> UTIL_LOGGER.setLevel(logging.WARNING)

Pandas¶

In addition, 2 classes have been taken verbatim from the Pandas dev team. The classes are used to create dict style objects that also allow for attribute-style access.
